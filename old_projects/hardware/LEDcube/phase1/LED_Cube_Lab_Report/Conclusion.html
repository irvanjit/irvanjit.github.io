to conclude

In conclusion, the LED board was successfully constructed and tested. It is now fully functional and ready to be further programmed in subsequent labs in the course. There was much to be learned from this project, such as how to build a functional three-dimensional display, concepts like jumping cables, and the use of smaller inexpensive ICs to increase the number of outputs when a very large number is required. There was also much to learn when solving problems  associated with the display, namely troubleshooting skills applied to hardware and software. This was a really interesting project.
