purpose

 
 

The purpose of this lab was to successfully program the previously constructed LED (Light Emitting Diode) Cube that has 4 levels, each with 4 rows and 4 columns of LEDs, such that once the cube was successfully programmed, the PICAXE chip would store the code and run it, independent of a connected computer, and would be able to make full use of the display’s three-dimensional layout of 64 LEDs to display various static and dynamic images of the user/programmer’s preference.
