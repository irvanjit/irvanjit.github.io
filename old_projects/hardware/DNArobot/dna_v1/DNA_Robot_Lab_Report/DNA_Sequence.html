Background Information:


These are the sequences used to program the DNA Robot for this lab report.


The DNA was obtained from DNA Sequences of four different chromosomes from the HUMAN GENOME.


The following chromosomes were used:


-Chromosome 1

-Chromosome 2

-X Chromosome

-Y Chromosome


An image was constructed from for each of these chromosomes. For each image, a sequence of 100 base pairs was used.


Bases Found in DNA:


A – Adenine

T – Thymine

G – Guanine

C – Cytosine


1 Base Pair contains 2 bases. Each base is represented in DNA code by its corresponding letter above.
