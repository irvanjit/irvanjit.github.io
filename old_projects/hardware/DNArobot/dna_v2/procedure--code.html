Procedure

  • Download the Proton IDE from here.
  • Install the Proton IDE.
  • Copy and paste this code into the editor window:
    Device 16F628A

    REMARKS    =On

    Config     WDT_OFF, BODEN_ON,INTRC_OSC_NOCLKOUT,LVP_OFF,CP_OFF, PWRTE_ON, MCLRE_OFF

    Dim MyCounter As Byte

    Dim     BasePair As Byte

    Symbol     L1 =     PORTA.2

    Symbol L2 =     PORTA.3

    Symbol L3 =     PORTA.0

    Symbol L4 =     PORTA.1

    ;TRISA=%01000000 ;Set Port "A", to output

    TRISA=0x00

    TRISB=0x00

    LOOP:

    For MyCounter = 0 To 62 Step 1

    Let     PORTA = %00000000

    BasePair = LRead DNA + MyCounter

    If BasePair =     "G" Then

    Let PORTA = %10100000

    DelayMS 1000

    EndIf

    If     BasePair =     "A" Then

    High     L2

    DelayMS 1000

    EndIf

    If     BasePair =     "T" Then

    High     L3

    DelayMS 1000

    EndIf

    If     BasePair =     "C" Then

    Let     PORTA = %01010000

    DelayMS 1000

    EndIf

    Next

    End

    DNA:

    LData     "A","T","C","G","A","G","C"

    LData "T","G","C","C","A","A","G"

    LData "A","T","A","C","C","C","A"

    LData "T","T","C","T","T","A","A"

    LData "A","A","C","A","T","G","A"

    LData "A","A","A","A","G","C","A"

    LData "G","C","T","G","A","C","C"

A brief code explanation:
  • The program configures PORTA and PORTB as outputs
  • Defines DNA code at the bottom of the program using LDATA statements
  • The program then iterates through the data using a FOR-NEXT-LOOP and analyzes it using IF-THEN-ELSE statements
  • If a certain letter is detected it outputs high to a certain pin, causing the motors to turn in certain way, moving the robot
  • DNA Code: Human Chromosome 1 (Human Genome Project) (source: http://www.gutenberg.org/etext/3501)
  • Compile the code.
  • Load the hex file onto the 16F628 microcontroller using MPLAB PICKit 2.
  • The code in the chip can be tested by using 4 LEDs to output from the first 2 and the last 2 pins of the chip
  • The LEDs should turn on and off at 1 second intervals in the order of the DNA code in the program
  • Install the chip on the circuit.
  • Follow the schematic included in the schematics section
  • The circuit requires 1 H-bridge per motor. Therefore, in total, there needs to be 2 H-bridges
  • A small pencil would have to be mounted in front of the robot, so that there is less resistance on the motors
