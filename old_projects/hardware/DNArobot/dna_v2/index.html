Introduction
The purpose of this lab is to use a robot to create an image based on DNA data gathered from actual organisms. DNA is composed of the nucleotides G, A, T and C. If we put this data into a program and tell the robot to move forward for G, left for A, right for T and backwards for C, we will be able to visualize this data. The end effect of this is to generate patterns which may help us to understand differences in the DNA of various organisms.
